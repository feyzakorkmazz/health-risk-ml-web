# Health Risk Analysis with Machine Learning and Web Dashboard

This project analyzes the relationship between smoking, COPD, and depression
using machine learning models and presents the results via a web-based dashboard.

## Technologies
- Python (Pandas, Scikit-learn, XGBoost)
- PHP, HTML, CSS, JavaScript
- MySQL
- Bootstrap

## Machine Learning Models
- Logistic Regression
- Random Forest
- XGBoost (final model)

## Dataset
CDC-based public health survey dataset  
(Preprocessed & sampled for performance)

## Web Application
- User-based health risk assessment
- Session & Cookie management
- Data visualization dashboards

## How to Run
1. Train models in `ml/notebooks`
2. Run web app via XAMPP
3. Import sample CSV into MySQL
